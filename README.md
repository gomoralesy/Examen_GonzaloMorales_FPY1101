# Examen_transversal_FPY1101
Examen de programación FPY1101

Este repositorio contiene el examen transversal

Nombre del proyecto: SISTEMA CRUD DE FITPASS
- Hecho por: Gonzalo Morales Yáñez, estudiante ing. en informática

- Este sistema CRUD tiene la función de buscar cupos de membresía por tipo de plan y también por rango de precio en los diccionarios planes e inscripciones, separando los planes que no tienen cupos disponibles y mostrando la información de interés para el usuario. También, cuenta con las funciones para actualizar precios de los planes en los diccionarios, agregar nuevos planes solicitando los datos al usuario y, finalmente, eliminar planes por código interacctuando con los dos diccionarios. Cuenta con una serie de funciones de validación para las entradas que se le solicitan al usuario que tienen la tarea de evitar la caída del programa y el ingreso correcto de los datos solicitados al usuario para el funcionamiento del mismo.
- Para ejecutar el programa, solo se debe ejecutar en el archivo .py porque el programa está sostenido en un solo archivo con sus funciones.

- Las funciones implementadas en el sistema son:
    1. Búsqueda de cupos por tipo de plan.
    2. Búsqueda planes por rango de precio.
    3. Agregar planes con datos ingresados por el usuario.
    4. Actualizar precio de planes almacenados en los diccionarios.
    5. Eliminar planes por código interactuando con los dos diccionarios.
    6. Funciones de validación para inputs del usuario.
